# Rekono Frontend

## Project setup
```
npm install
```

### Configuration
The following properties can be configured in the `rekono/frontend/.env` file:

- `VUE_APP_DEFECTDOJO`: Enable or disable Defect-Dojo integration features in the frontend
- `VUE_APP_DEFECTDOJO_URL`: Defect-Dojo URL
- `VUE_APP_TELEGRAM_BOT`: Name of the Telegram Bot to be displayed in the UI

Please, review the [Rekono documentation](../../README.md#configuration) to see all the alternatives.

npm run serve
